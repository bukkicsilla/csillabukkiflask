Main Purpose:

Actors and actresses with movie titles can be stored in the library. Titles with the same name should be stored in a slightly different way. For example: The movie Jane Eyre exists with release years: 1943, 1997, 1983, 1996, 2006, 2011. So it is recommended to save them for example as Jane Eyre 1943, Jane Eyre 1997 and so on. It is advisable to do the same with identical actor and actress names.

-------------------------------------------------------

Menu: In the lower right corner on phones and in the upper right corner on tablets there is a menu with options: new, search, delete, help, settings.

New: An actor or an actress with a movie title should be given. These fields are mandatory. Next a photo can be choosen for the actor or actress, which is optional. The photo can be choosen or modified later on the last screen.

Search: An actor/actress - movie pair can be found or you can search for an actor/actress or movie title by leaving the other field empty. In that case there will be given a list of actors and actresses for a movie title, and a list of movie titles for an actor or actress.

Delete: A specific actor/actress - movie pair can be deleted. A word completion feature can help to find a pair. All pairs can be deleted at the same time by typing * in both fields. Help: There are 2 helping descriptions. One of them is with animation and the other one is without. Between the two modes it can be switched at the Help Animation

Settings: In the BASIC SETTING the Auto Back up means that if it is set to true the data will be kept after deleting and reinstalling the app. Another feature is the Sharing enabled option. If it is set to true, an actor/actress - movie pair with photos can be shared with others.

In the MORE SETTING if the Image Animation is set to true, an animation of photos, previously stored in the library, is shown in the start screen. The Help Animation is discussed under Help. The background color can be choosen with Color and the font family with Font.

-------------------------------------------------------

Actors: To get the list of actors and actresses, the ACTORS button has to be choosen on the start screen. On the next screen the order of the list can be selected BY NAME (alphabetical ascending order) or BY BIRTH (numerical descending order) .

Movies: To get the list of movie titles, the MOVIES button has to be choosen on the start screen. On the next screen the order of the list can be selected BY TITLE (alphabetical ascending order) or BY BIRTH (numerical descending order) .

-------------------------------------------------------

Actor Movie Pair:

This pair is on the last screen in the app. It shows the details about the actor or actress and the movie.

Image: The upper image should show the actor/actress and the lower one should show an image related to the movie title. To set or change an image, long press on the image or image frame. To view an image in full screen, short press on the image.

Menu: In the upper right corner there is a menu with options: edit, share, delete.

Edit: It opens a new screen, where the details (except actor/actress name and movie title) can be set or modified. Links to the actor or actress, and to the movie can be added by copying URLs into the fields Link to Actor or Actress: and Link to Movie: respectively.

Share: The actor or actress and the movie title with the accompanying images can be sent via Skype, Gmail, Drive, etc.

Delete: The actor or actress and the movie title pair can be deleted.

A button: and M button: If links to the actor or actress, or to the movie were set in the Edit screen, they can be opened in the browser by using the letter icons A and M respectively.
